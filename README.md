## readme-generator

## Purpose
An application using node.js that takes user input and makes a README.md file for the users repository.

## Built With
* JavaScript
* Node.Js
* use of ES6

## Usage
Need to install from command line
* npm init
* npm install inquirer
* node index.js to run app
